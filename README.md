- **Technical Skills**: HTML, CSS, JavaScript, React.js, Hardware Repair, Problem Solving

- **Portfolio Projects**: Ready-to-customize section for showcasing work

## How to Use
1. Open `cover-letter-template.md`
2. Replace all **[bracketed placeholders]** with job-specific details
3. Customize portfolio projects section with relevant work
4. Tailor skills and content to match the job description
5. Export to PDF or Word format for submission
